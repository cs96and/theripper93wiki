# Token Flip

Switch between different token images and actors on a single token.

## How to use

<ImageButBetter src="/../public/images/Token-Flip/Tfconfig.png" alt="Token Flip Configuration Button" width={2000} height={2000} className={styles.wikiimage}/>

### Configuring the different token faces

- Open the token configuration
- Click on the Configure Token Flip button
- Click the Add new face button to add faces
- Configure image, actor and scale as needed
- Close the window
<ImageButBetter src="/../public/images/Token-Flip/Tfwindow.png" alt="Token Flip Configuration Window" width={2000} height={2000} className={styles.wikiimage}/>

### Switching between faces

There are two ways to flip to a different face

#### Using the token HUD


Left click on the Token flip
button to cycle to the next face

#### Using the Token Flip menu
<ImageButBetter src="/../public/images/Token-Flip/Tfhud.png" alt="Token Flip button in the token HUD" width={2000} height={2000} className={styles.wikiimagesmall}/>
<ImageButBetter src="/../public/images/Token-Flip/Tfhudmenu.png" alt="Token Flip HUD Menu" width={2000} height={2000} className={styles.wikiimagesmall}/>


Left click
and hold the left mouse button down on the token flip button to bring up
the token flip menu, then click on the desired face.
