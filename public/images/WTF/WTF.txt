Provides tooltips that indicate which module is adding which option.

== How to use ==
Simply hover over a setting in an object's configuration window to display the corresponding module

== Configuration ==
In the module settings you can chose if you want to enable it for players as well, the position of the tooltip, and, if you are an advanced user, you can also show the data path for the flag in the tooltip.
